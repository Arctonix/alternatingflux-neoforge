CHUNK 3 — CRAFTING RECIPES (survival-ready)
============================================
Adds the four recipes so everything is obtainable in survival, ported from the
original mod's shapes to modern data-driven JSON with current IE tags/items.

RECIPES
  - wire_constantan : constantan plate + wirecutter (shapeless) -> 4 Constantan Wire.
                      Mirrors IE's own wire_lead recipe exactly; the wirecutter is
                      DAMAGED, not consumed (vanilla tool-remainder behaviour).
  - wirecoil_af     : constantan wire + HOP graphite dust + wooden rod -> 4 AF Wire Coil.
  - connector_af_relay : aluminum ingot + insulating/slag glass -> 8 AF Wire Relay
                         (same shape as IE's HV relay).
  - connector_af_transformer : HV connector + AF relay + HV coil + constantan + iron
                               -> 1 AF Transformer (built on an HV connector, like the
                               original).
  Plus a tag (c:wires/constantan) so the coil recipe can reference constantan wire.

APPLY:
  cd ~/Downloads
  unzip -o chunk3.zip -d ~/Documents/"AF mod"/afmod
  cd ~/Documents/"AF mod"/afmod
  ./gradlew runClient

TEST (in a SURVIVAL world, or creative with a crafting table):
  - Constantan plate + wirecutter -> Constantan Wire (cutter survives, loses durability).
  - Constantan wire x? + HOP graphite + wooden rod in the coil pattern -> AF Wire Coil x4.
  - Aluminum + glass in the relay pattern -> AF Wire Relay x8.
  - HV connector + AF relay + HV coil + constantan + iron -> AF Transformer.
  - Check JEI/EMI if you have it: search "AF" and all four should show recipes.
