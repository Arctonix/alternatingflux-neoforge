CHUNK 1 — AF RELAY — apply instructions
========================================

This adds the AF Wire Relay: the wire-to-wire connector that AF wire attaches to.
AF is transmission-only, so there is NO machine connector — only this relay.

HOW TO APPLY (from your project root: ~/Documents/"AF mod"/afmod)

1. Unzip this over your project. From the folder where you downloaded chunk1.zip:
     unzip -o chunk1.zip -d ~/Documents/"AF mod"/afmod
   (-o overwrites AlternatingFlux.java with the new version — that's intended.)

2. Fix the block texture folder to the singular atlas path (same fix we did for items).
   The relay's texture must live in textures/block/ (singular), not textures/blocks/:
     cd ~/Documents/"AF mod"/afmod/src/main/resources/assets/alternatingflux
     mv textures/blocks textures/block
   (This also pre-positions the transformer + passthrough textures for Chunk 2.)

3. Build and run:
     cd ~/Documents/"AF mod"/afmod
     ./gradlew runClient
